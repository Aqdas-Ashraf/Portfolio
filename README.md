# 🎬 Aqdas Ashraf Portfolio

A modern and responsive video editing portfolio showcasing my editing work, skills, and projects.

## 🌐 Live Demo

https://aqdas-ashraf.github.io/Portfolio/

## 📌 About

This portfolio is designed to showcase my video editing skills and creative projects. It includes short-form and long-form editing samples with a clean and modern user interface.

## ✨ Features

- Responsive Design
- Modern UI
- Smooth Animations
- Short Form Video Showcase
- Long Form Video Showcase
- Mobile Friendly
- Fast Loading

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Swiper.js

```
Portfolio/

```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Aqdas-Ashraf/Portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
